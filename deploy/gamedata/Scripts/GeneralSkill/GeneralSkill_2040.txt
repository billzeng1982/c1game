--[[
名称：干戚之舞
类型：主动技能
描述：典韦摆出轮枪阵型，可以对所有方向的敌人造成枪击
--]]
local ThisScriptId = 2040;

--技能诞生处理
local function runBorn(LuaDataAgt)
	-- 设置公式
	LuaDataAgt:SetFormula(VALUE_CHG_TYPE.CHG_HP_GENERALSKILL, ThisScriptId, 0);
	--LuaDataAgt:SetFormula(VALUE_CHG_TYPE.CHG_HP_SKILLCHEATS, ThisScriptId, 0);
	
	local lifeTime = LuaDataAgt:GetActiveSkillBaseValue(2);
	if LuaDataAgt:IsHaveActiveSkillCheats(1) then     --秘籍1：增加时长
	    lifeTime = lifeTime * 1.1;
	end
	
	LuaDataAgt:SetShareValue(0, lifeTime);		-- 生命周期
	
	local damageCd = 1.5;
	LuaDataAgt:SetShareValue(1, damageCd);		-- 伤害CD
	
	LuaDataAgt:SetShareValue(2, 0.1);			-- 首次伤害	
	LuaDataAgt:SetShareValue(3, 40);			-- BuffID
end

--技能条件判断
local function checkCond(LuaDataAgt, AIId)
	--小地图技能目标提示标志
	LuaDataAgt:ClearAllSkillTarget();
	--没有技能框 清除其它技能框
	LuaDataAgt:ClearSkillRange();
	local targetList = LuaDataAgt:FindTargetEx(TARGET_GROUP.ENEMY, SHAPE_TYPE.CIRCLE, 0, 0, 15, 0);
	--AI操作需要周围15码内至少1人，玩家操作无要求
	if (AIId == 0) then
		
	else
		if targetList.Count > 0 then
			return true;
		else
			return false;
		end
	end
	return true;
end

--找对象，上面条件满足后执行
local function findTarget(LuaDataAgt)
	--找之前先清一次
	LuaDataAgt:ClearTarget();
	
	--设置技能目标为自己，表示该技能有目标，生效会延迟
	local targetList = LuaDataAgt:FindTarget(TARGET_GROUP.NONE);
end

--技能生效，数值处理
local function takeEffect(LuaDataAgt)
	LuaDataAgt:InterruptSiege();
	LuaDataAgt:AddBuff(ThisScriptId, LuaDataAgt, nil);--加暴击
	LuaDataAgt:AddBuff(564, LuaDataAgt, nil);--加速
	local TarPos = LuaDataAgt:GetCurPos();
	LuaDataAgt:PlayFX("Prefabs/BasicEffect/3D_CharEff_BaoJiShangSheng", LuaDataAgt, TarPos, TarPos, 0, true); -- 延迟0播该特效
end

--技能公式
local function doFormula(LuaDataAgt, iFormulaType, iFormulaPara, rSource, rTarget, iDamageRef)
	local iRet = 0;
	
	local fStrAtk = rSource:GetStrAtk(VALUE_TYPE.CURRENT);
	local fStrDef = rTarget:GetStrDef(VALUE_TYPE.CURRENT);
	
	local fSpearDamageBase = rSource:GetActiveSkillBaseValue(1);
	local fSpearDamageMin = fSpearDamageBase;
	local fSpearDamageRatio = rSource:GetActiveSkillRateValue(1);
	local fCounterDamageMin = fSpearDamageBase;	
	local fCounterDamageBase = fSpearDamageBase ;
	local fCounterDamageRatio = fSpearDamageRatio ;
	
    if iDamageRef == 3 then
		--枪击
		iRet = math.max(((fSpearDamageBase + (fStrAtk - fStrDef) * fSpearDamageRatio)), fSpearDamageMin);
	elseif iDamageRef == 5 then
	   --迎击
	   iRet = math.max(((fCounterDamageBase + (fStrAtk - fStrDef) * fCounterDamageRatio)),fCounterDamageMin);
	end
	
	return -iRet;
end


GeneralSkill_2040 = {
	func_runBorn = runBorn;
	func_checkCond = checkCond;
	func_findTarget = findTarget;
	func_takeEffect = takeEffect;
	func_doFormula = doFormula;
}
