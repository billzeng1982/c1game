--[[
名称：金刚护体
类型：被动技能
描述：周泰在15s内未受到伤害，则获得一个生命值为300的护盾
--]]
local ThisScriptId = 5244;

--技能诞生处理
local function runBorn(LuaDataAgt)
	-- 添加血量过滤处理
	LuaDataAgt:SetFilter(FILTER_TYPE.HP, TARGET_GROUP.NONE, 0);
	
	LuaDataAgt:SetFilter(FILTER_TYPE.TRIG, TARGET_GROUP.NONE, 0);
	if not IsOnServer then
		LuaDataAgt:SetShareTimer(0, 15);
	end
end

--技能条件判断
local function checkCond(LuaDataAgt)
	return true;
end

--找对象，上面条件满足后执行
local function findTarget(LuaDataAgt)
	--找之前先清一次
	LuaDataAgt:ClearTarget();
end

--技能生效，数值处理
local function takeEffect(LuaDataAgt)
	if IsOnServer then
		LuaDataAgt:SetSkillEnable(false);
		return;
	end
	
	local shareTimer = LuaDataAgt:GetShareTimer(0);
	if shareTimer < 0 then
		LuaDataAgt:SetShareTimer(0, 15);
		--LuaDataAgt:AddBuff(ThisScriptId, LuaDataAgt, nil);
	end
end

--Skill过滤处理
local function doFilter(LuaDataAgt, iFilterType, rFilterOwner, FilterValue)
	-- Filter所有者是否是发起对象
	local IsSource = rFilterOwner:Equal(FilterValue:GetSource());
	
	if iFilterType == FILTER_TYPE.HP then
		local iHp = FilterValue.m_fFoo1;
		if not IsSource and iHp < 0 then
			if not IsOnServer then
				LuaDataAgt:SetShareTimer(0, 15);
			end
		end	
	end
	
	if iFilterType == FILTER_TYPE.TRIG and FilterValue.m_iValueChgType == VALUE_CHG_TYPE.CHG_TRIG_BUFF_ADD_END then
		local dwBuffId = FilterValue.m_fFoo1;
		if dwBuffId == 9001 then
			FilterValue.m_fFoo1 = ThisScriptId;
		end	
	end
	
	return FilterValue;
end


PassiveSkill_5244 = {
	func_runBorn = runBorn;
	func_checkCond = checkCond;
	func_findTarget = findTarget;
	func_takeEffect = takeEffect;
	
	func_doFilter = doFilter;
}
